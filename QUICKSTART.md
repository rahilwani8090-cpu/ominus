# Ominus AI - Quick Start Guide

## ⚡ 2-Minute Setup

### 1. Open Browser
Open `index.html` or run:
```bash
python -m http.server 8000
```

### 2. Chat
- Type message → Press Enter
- Model: Llama 3.3 (pre-configured, free)

### 3. Done! 🎉

---

## 🎤 Voice (30 seconds)

### Enable Voice Input
1. Click 🎤 microphone button
2. Speak (English or हिंदी)
3. Message auto-sends

### Enable Voice Output
1. Settings ⚙️ → Voice
2. Check "Auto-play voice responses"
3. AI responds with voice

---

## ⚙️ Automation (1 minute)

### Quick Automation
1. Have a conversation
2. Click "Automate" button
3. Pick one:
   - 📝 Summary
   - ✅ Extract Tasks
   - 📄 Blog Post
   - ✉️ Email
   - 📱 Social Media
   - ⚖️ Compare Models

### Result
- Copy text
- Download file
- Listen with voice

---

## 🎨 Brand (1 minute)

### Customize
1. Settings ⚙️ → Branding
2. Change name
3. Pick color
4. Upload logo
5. Save

### Result
- Your brand everywhere
- Share branded link
- Professional appearance

---

## 🔑 API Keys (Optional)

### Add Your Own Keys

1. Settings ⚙️ → API Keys
2. Paste your keys:
   - Groq (optional - built-in)
   - Gemini
   - OpenAI
   - Claude
   - xAI
3. Save

### Get Keys
- [Groq](https://console.groq.com/keys) (FREE)
- [Gemini](https://makersuite.google.com/app/apikey)
- [OpenAI](https://platform.openai.com/api-keys)
- [Claude](https://console.anthropic.com/settings/keys)
- [xAI](https://console.x.ai)

## 🎯 Common Tasks

### Summarize Meeting
1. Chat about meeting
2. Click Automate → Summary
3. Copy summary
4. Done!

### Generate Blog Post
1. Chat about topic
2. Click Automate → Blog Post
3. Export as .md
4. Add to blog
5. Done!

### Compare AI Models
1. Ask a question
2. Click Automate → Compare
3. See 3 different answers
4. Pick best one
5. Done!

### Send to Multiple Languages
1. Set Voice Output → Hindi
2. Toggle Auto-play voice
3. Chat in English
4. AI responds in Hindi with voice
5. Team understands both!

### Share with Team
1. Customize branding
2. Share link (works offline too)
3. Team sees YOUR brand
4. No server needed
5. Done!

## 🆘 Troubleshooting

### Voice not working?
- Use Chrome/Edge/Safari
- Check microphone permissions
- Try browser TTS first

### Model not responding?
- Add API key in Settings
- Check key is valid
- Try different model

### Can't save settings?
- Check browser localStorage enabled
- Clear cache & try again
- Use different browser

### Automation not showing?
- Check checkbox in Settings → Automation
- Have a conversation first
- Try simpler prompt
