# 🎉 GROQ API SETUP - READY TO USE

## What You Need To Do (5 Minutes)

### Step 1: Get Your Free Groq API Key
1. Go to: https://console.groq.com/keys
2. Sign up (free, no credit card needed)
3. Copy your API key (starts with `gsk_...`)

### Step 2: Add Key to Ominus (Choose One)

**Option A: Edit app.js (Recommended)**
```javascript
// In app.js, line 9, replace:
defaultGroqKey: 'gsk_YOUR_GROQ_API_KEY_HERE',

// With your actual key from console.groq.com/keys:
defaultGroqKey: 'gsk_YOUR_API_KEY_GOES_HERE',
```

**Option B: Use Settings UI**
1. Open the app
2. Click ⚙️ Settings
3. Go to "API Keys" tab
4. Paste your Groq key
5. Click "Save All Settings"

### Step 3: Start Chatting! 🎉
- Open the app
- Start typing your message
- Instant AI responses!

## 🆘 If Something Doesn't Work

### "No response" after typing
- Check console (F12) for errors
- Verify API key is correct
- Try refreshing the page

### "Rate limited" error
- Wait 5 minutes or
- Upgrade to Groq Pro or
- Add another provider's key

### Voice not working
- Chrome or Edge browser only
- Check microphone permissions
- Check microphone volume

### Models not available
- Check your Groq key is valid
- Verify internet connection
- Try refreshing
